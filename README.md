# Microblog

* Configuring the **docker-compose-template.yml** and renaming it in **docker-compose.yml**
```dockerfile
    environment:
      - HOST=***
      - DBNAME=***
      - USER=***
      - PASSWORD=***
      - STATIC_PATH=/app/app/static
```

* Run
```shell script
docker-compose up
```
